## Tugas 6 Pertemuan 8
Penjelasan Proses Passing Data
Pada tugas Flutter ini, proses passing data dilakukan dari halaman form input (DataForm) menuju halaman tampilan hasil (DataDetail).

Di halaman DataForm, pengguna mengisi beberapa data seperti nama, NIM, dan tahun lahir melalui widget TextField. Nilai-nilai yang dimasukkan tersebut ditampung menggunakan TextEditingController. Ketika tombol Simpan ditekan, data dari controller dikirim ke halaman berikutnya menggunakan Navigator dan MaterialPageRoute.
Data yang dikirim disertakan sebagai parameter konstruktor dari kelas DataDetail. Halaman DataDetail kemudian menerima data tersebut melalui variabel final yang dideklarasikan di dalam kelas, dan menampilkannya dalam bentuk teks atau paragraf.
Dengan cara ini, Flutter memungkinkan komunikasi antar halaman (widget) dengan mudah menggunakan konsep passing data melalui konstruktor dan navigasi antar layar.
